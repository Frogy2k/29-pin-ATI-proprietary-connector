# 29 pin ATI proprietary connector
## Description:
Reversed pinout of the 29 pin ATI connector for some ATI All-in-Wonder cards.  
## Supported Cards:
- ATI All-in-Wonder 9600  
## Pinout
| GND |   |       | GND |   |
|     |   |       | GND |   |
|     |   |       |     |   |
|     |   | HSync | GND | B |
|     |   | VSync | GND | G |
|     |   | GND   | R   |
